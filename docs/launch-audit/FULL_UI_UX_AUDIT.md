# Full UI/UX Audit – Dynasty Degens

This section examines the user interface and experience of the live app, focusing on the Skids Get Beat league (Dynasty, Superflex).  Each finding includes an ID, severity, category, affected page/component, description, rationale, and recommended fix.  Severity levels: **Critical**, **High**, **Medium**, **Low**, **Polish**.

### Additional observations

- **Owner Intel Lab** and **Trade Reaper** include complex charts and metrics.  While these features are impressive, they may overwhelm novices.  Consider progressive disclosure or tooltips explaining each metric.
- The bottom navigation bar is helpful but hidden behind the overlay until the league fully loads; this may confuse users who need to cancel loading.
- There is no global search or help icon.  New users may struggle to understand all the features without guidance.
- Many icons lack tooltips; adding them would improve clarity.
