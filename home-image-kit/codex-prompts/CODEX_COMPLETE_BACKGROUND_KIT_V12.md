# Dynasty Degens Complete Background Kit v12

Use this as the full replacement background kit.

The previous partial kits caused too much folder juggling. This kit includes everything:
- viewport hero backgrounds
- long/page backgrounds
- mobile-inline DD/player backgrounds
- crisp electric floor continuation backgrounds

Production path:
`public/brand/backgrounds/`

Use:
- `viewport/jpg/*`
- `long/jpg/*`
- `mobile-inline/jpg/*`
- `continuation/jpg/*`
- `code/complete-backgrounds-v12.css`

Rules:
1. Desktop/tablet/ultrawide hero images should keep the helmet top-locked and preserve the right-side player/flames.
2. Mobile portrait should use the mobile-inline assets where the DD sits behind/above the player, with the helmet overlapping some of the DD.
3. Long pages should start with the hero crop and blend into the crisp electric floor.
4. Expanded tables/open sections should use the crisp floor continuation backgrounds.
5. The continuation should have bright cyan/orange grid lines and sparkles.
6. Do not use old dull continuation assets.
7. Do not change app logic, Sleeper logic, routes, tabs, tables, or copy.

After implementing:
- Verify desktop wide.
- Verify ultrawide.
- Verify tablet landscape and portrait.
- Verify mobile portrait.
- Verify opened tables continue into the crisp floor background.
